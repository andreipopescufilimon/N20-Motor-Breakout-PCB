# N20 Motor Breakout with DRV8212 4A Driver

This N20 Motor Breakout is built around the DRV8212 motor driver. It is a compact tiny board that fits directly on the motor. The DRV8212 delivers strong 4A output current while staying cool and efficient, so the motor can handle heavy loads and fast direction changes without needing external driver hardware.

## Key Features
- DRV8212 integrated H bridge motor driver  
- Up to 4 A peak output current  
- Designed for N20 brushed DC gear motors  
- Four wire interface: VCC, GND, DIR, PWM  
- Works with 3.3 V and 5 V logic  
- PWM based speed control  
- Direction control with a single pin  

## Pin Functions
The breakout uses a simple four pin interface:

- **VCC**: Motor supply voltage input  
- **GND**: Ground  
- **DIR**: Sets motor direction. High or low selects the rotation direction  
- **PWM**: Speed control input. A PWM signal controls how much power is sent to the motor
